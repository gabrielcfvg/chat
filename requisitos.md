# Requisitos para o chat

- Reações
- Horário de envio da mensagem
- Mostrar quando a mensagem já foi enviada ou se está enviando (no discord, por exemplo, a mensagem fica transparente até o envio)
- Mostrar quais pessoas já viram, por exemplo, com a foto de perfil dos que viram embaixo ou na direita da mensagem (Caso a quantidade de pessoas que viram ultrapasse o limite, terá um símbolo de "...", onde quando vc clica aparecem todas as pessoas que viram)
- Mensagens em MarkDown
- Mostrar se alguém está online (com a opção de mudar seu status para offline mesmo quando você está online)
- Não pertube
- Bloquear
- Salas de chat
- Figurinhas são legais
- Opção de envio de vídeo (compactado é claro)
- Talvez um status? As pessoas gostam disso né?
- API simples (em python pra agradar os jurados)
- Versão de linha de comando??
- Figurinhas personalizadas
- Vasta quantidade de temas e opções, afogar o usuário em configurações dá uma impressão de imensidão e garante que o usuário nunca irá enjoar do estilo
- Mas é claro que não pode faltar mudar a imagem de fundo do chat
